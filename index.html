Introducción: Backend vs Frontend

🎨 Frontend (Cliente)

¿Qué es? La parte que el usuario ve y con la que interactúa directamente.

Ejemplo bancario: El formulario que un cliente llena para solicitar un préstamo, los botones que hace clic, las tablas que visualiza.

Tecnologías: HTML, CSS, JavaScript (en el navegador)

En GAS: HTML Service, diálogos, sidebars, webapps

⚙️ Backend (Servidor)

¿Qué es? La parte que procesa datos, ejecuta lógica y se conecta con bases de datos (todo invisible para el usuario).

Ejemplo bancario: El script que calcula la tasa de interés, verifica el saldo, actualiza la hoja de cálculo y envía el email de confirmación.

Tecnologías: Google Apps Script, Python, Java, etc.

En GAS: Funciones .gs, SpreadsheetApp, GmailApp, DriveApp

Analogía Bancaria (Restaurante): El Frontend es el menú y el camarero que toma tu pedido. El Backend es la cocina donde preparan la comida, con sus fogones, neveras y cocineros. El cliente ve el menú (Frontend), pero no la cocina (Backend).
Ciclo Completo en GAS:
  1. Frontend: El usuario hace clic en un botón o llena un formulario
  2. Comunicación: Los datos viajan al backend mediante google.script.run
  3. Backend: La función .gs procesa los datos (lee/escribe en Sheets, envía emails, etc.)
  4. Respuesta: El backend devuelve el resultado al frontend
  5. Frontend: Muestra el resultado al usuario (confirmación, error, etc.)

Backend: El Corazón del Sistema (Lo más básico)

El backend en GAS son funciones escritas en archivos .gs que se ejecutan en los servidores de Google. Aquí tienes ejemplos muy básicos que usaremos con los formularios:

Backend .gs - Funciones Básicas 
// Función para saludar (usada en ventana de diálogo)
function saludarCliente(nombre) {
  return "Hola " + nombre + ", gracias por usar nuestros servicios bancarios.";
}

// Función para registrar una operación (usada en caja HTML)
function registrarOperacion(tipo, monto, cliente) {
  let ss = SpreadsheetApp.getActiveSpreadsheet();
  let hoja = ss.getSheetByName("Operaciones");
  
  if (!hoja) {
    hoja = ss.insertSheet("Operaciones");
    hoja.appendRow(["Fecha", "Cliente", "Tipo", "Monto"]);
  }
  
  hoja.appendRow([new Date(), cliente, tipo, monto]);
  return "Operación registrada correctamente. Monto: " + monto + " EUR";
}

// Función para filtrar por columna (usada en sidebar)
function filtrarPorColumna(columna, valor) {
  let ss = SpreadsheetApp.getActiveSpreadsheet();
  let hoja = ss.getActiveSheet();
  let datos = hoja.getDataRange().getValues();
  
  let resultados = [];
  let cabeceras = datos[0];
  
  for (let i = 1; i < datos.length; i++) {
    if (datos[i][columna].toString().toLowerCase().includes(valor.toLowerCase())) {
      resultados.push(datos[i]);
    }
  }
  
  return {
    cabeceras: cabeceras,
    datos: resultados,
    total: resultados.length
  };
}

// Función para exportar a CSV (usada en sidebar)
function exportarACSV(datos, nombreArchivo) {
  let csvContent = "";
  
  // Convertir datos a formato CSV
  datos.forEach(function(fila) {
    csvContent += fila.join(",") + "\n";
  });
  
  // Crear archivo en Drive
  let archivo = DriveApp.createFile(nombreArchivo + ".csv", csvContent, MimeType.CSV);
  return archivo.getUrl();
}

// Función para guardar desde WebApp (usada en formulario externo)
function guardarSolicitudWebApp(data) {
  let ss = SpreadsheetApp.openByUrl("URL_DE_TU_SPREADSHEET"); // Cambiar por URL real
  let hoja = ss.getSheetByName("Solicitudes");
  
  if (!hoja) {
    hoja = ss.insertSheet("Solicitudes");
    hoja.appendRow(["Fecha", "Nombre", "Email", "Producto", "Monto"]);
  }
  
  hoja.appendRow([new Date(), data.nombre, data.email, data.producto, data.monto]);
  
  // Enviar confirmación por email
  GmailApp.sendEmail(
    data.email,
    "Solicitud Recibida - Banco Ejemplo",
    "Hola " + data.nombre + ",\n\nHemos recibido tu solicitud de " + data.producto +
    " por " + data.monto + " EUR. Un asesor se pondrá en contacto contigo.\n\nGracias."
  );
  
  return "Solicitud guardada y email enviado";
}
📌 Nota: Estas funciones son la "cocina" (backend). Ahora veremos los 4 tipos de "menús" (frontend) para interactuar con ellas.

Tipo 1: Ventana de Diálogo (UI)

Formulario Tipo 1 - Ventana Modal

📱 Ventana de Diálogo con Prompt

La forma más simple de interacción: una ventana emergente que pide un dato al usuario.

Ejemplo Bancario: Saludar a un Cliente
Backend .gs - Función para Diálogo 
function mostrarDialogoSaludo() {
  // Esta función muestra el diálogo y llama al backend
  let ui = SpreadsheetApp.getUi();
  
  let respuesta = ui.prompt(
    'Saludo Bancario',
    'Por favor, introduce tu nombre:',
    ui.ButtonSet.OK_CANCEL
  );
  
  if (respuesta.getSelectedButton() == ui.Button.OK) {
    let nombre = respuesta.getResponseText();
    let mensaje = saludarCliente(nombre); // Llama a la función backend
    ui.alert('Mensaje', mensaje, ui.ButtonSet.OK);
  }
}

// La función saludarCliente ya la definimos antes
function saludarCliente(nombre) {
  return "Hola " + nombre + ", gracias por usar nuestros servicios bancarios.";
}

      .setHeight(350);
  SpreadsheetApp.getUi()
      .showModalDialog(html, 'Registrar Nueva Operación');
}
